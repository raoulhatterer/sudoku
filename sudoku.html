2 Où le joueur doit-il prendre les chiffres à placer sur la grille?

Chaque fois que le joueur doit prendre un symbole à placer sur la grille, il va le piocher dans un des 9 sacs contenant chacun des symboles identiques.

2.1 Sacs de chiffres (restant à placer sur la grille)

Un sac contient des symboles identiques. 

 1: class sac:
 2:     """
 3:     Classe représentant un sac. Un sac contient des symboles identiques.
 4: 
 5:     Attributs:
 6:     ---------
 7:     symbole : symbole placé dans le sacs
 8:     cardinal : nombre de symboles que le sac contient
 9:     """
10:     def __init__(self, symbole):
11:         """
12:         Constructeur de sac contenant 9 symboles identiques
13:         """
14:         self.symbole = symbole
15:         self.cardinal = 9  # nombre d'éléments dans le sac
16: 
17:     def __repr__(self):
18:         """
19:         Représentation officielle d'un sac lorsque l'on tape son nom dans
20:         l'interpréteur.
21: 
22:         Le contenu du sac est renvoyé.
23:         """
24:         if self.cardinal == 0:
25:             return "sac vide"
26:         elif self.cardinal == 1:
27:             return "contient 1 chiffre {}".format(self.symbole)
28:         else:
29:             return "contient {} chiffres {}".format(self.cardinal,
30:                                                     self.symbole)

2.2 La pioche

La pioche est composée de 9 sacs contenant chacun des symboles identiques tous différents: 1 sac avec que des "1", un autre avec que des "2", etc.

  • À l'initialisation de la pioche, chaque sac contient 9 symboles identiques.
  • À l'initialisation de la grille, un certain nombre de ces symboles est prélevé dans la pioche pour les placer sur la grille.

  • Au cours du jeu, tant qu'il reste des symboles dans la pioche, le joueur peut en piocher pour les placer sur la grille. 

    33: class pioche():
34:     """
35:     Classe représentant 9 sacs contenant chacun des symboles identiques
36:     tous différents (1 sac avec que des "1", un autre avec que des "2", etc.)
37: 
38:     À l'initialisation de la grille, un certain nombre de ces symboles est
39:     prélevé pour les placer sur la grille.
40:     Au cours du jeu, tant qu'il reste des symboles dans un sac, le joueur peut
41:     en piocher pour les placer sur la grille.
42: 
43:     exemple:
44:     -------
45:     ma_pioche = pioche()
46:     print(ma_pioche)         # affiche la pioche (contenu des 9 sacs de pioche)
47:     print(ma_pioche.contenu) # affiche la pioche sous forme de liste
48:     print(ma_pioche[1])      # affiche le contenu du premier sac de pioche
49: 
50: 
51:     """
52:     NBR_SACS = 9
53: 
54:     def __init__(self):
55:         """
56:         Initialisation d'une liste contenant 9 sacs
57: 
58:         chaque sac contenant chacun 9 symboles identiques
59:         """
60:         self.contenu = [sac(numéro) for numéro in range(1, self.NBR_SACS+1)]
61: 
62:     def __getitem__(self, index):
63:         """
64:         Permet d'obtenir le contenu d'un sac dans la pioche avec:
65:         ma_pioche[index] # où index est compris entre 1 et NBR_SACS.
66:         """
67:         return self.contenu[index-1]
68: 
69:     def __setitem__(self, index, sac):
70:         """
71:         Permet d'écrire un sac dans la pioche.
72: 
73:         exemple:
74:         -------
75:         ma_pioche[9] = sac(9)
76:         """
77:         if index < self.NBR_SACS+1:
78:             self.contenu[index-1] = sac
79:         else:
80:             raise IndexError()
81: 
82:     def __len__(self):
83:         return self.NBR_SACS
84: 
85:     def __repr__(self):
86:         """
87:         Représentation de la pioche.
88: 
89:         Le contenu des 9 sacs est renvoyé.
90:         """
91:         affichage = ""
92:         for sac in self.contenu:
93:             affichage += sac.__repr__() + "\n"
94:         return affichage

3 Cases et grille

La grille comporte 81 cases.

3.1 Définissons ce qu'est une case

 97: class case:
 98:     """
 99:     Classe représentant une case.
100: 
101:     Une case non vide a un `contenu`, le symbole qui est affiché quand on tape
102:     le nom de la case dans l'interpréteur.
103:     Une case vide à des prétendants (valeurs possibles de la case).
104:     Une case a des cases cousines qui sont soit dans la même ligne, soit dans
105:     la même colonne soit dans le même carré (3 x 3).
106: 
107:     exemple:
108:     -------
109:     ma_case = case()
110:     print(ma_case)
111:     """
112:     def __init__(self):
113:         self.contenu = None
114:         self.pretendants = [1, 2, 3, 4, 5, 6, 7, 8, 9]
115:         self.cousines = None
116: 
117:     def __repr__(self):
118:         """
119:         Affichage d'une case.
120: 
121:         Lorsque l'on tape son nom dans l'interpréteur
122:         son `contenu` est affiché.
123:         """
124:         if self.contenu is None:
125:             return "0"    # "⛶"
126:         else:
127:             return "{}".format(self.contenu)

3.2 Construisons la grille

129: class grille:
130:     """
131:     Classe représentant une grille de 9 x 9 cases.
132: 
133:     Chacune des 81 cases est accessible via un index allant de 0 à 80.
134: 
135:     exemple:
136:     -------
137:     ma_grille = grille()
138:     print(ma_grille)            # affiche la grille 9 x 9 cases
139:     print(ma_grille.contenu)    # affiche la grille sous forme de liste
140:     print(ma_grille[0]) # affiche la première case (son index est 0)
141: 
142:     options:
143:     -------
144:     Affichage en couleur (fonctionne uniquement dans le Terminal pas dans la
145:     console Python): ma_grille.affichage_en_couleur = True
146:     """
147: 
148:     COTÉ = 9
149:     NBR_CASES = COTÉ * COTÉ
150: 
151:     def __init__(self):
152:         liste_cases = list()
153:         for index in range(self.NBR_CASES):
154:             une_case = case()
155:             cousines_en_ligne = [i for i in range(
156:                 self.COTÉ*self.getligne(index),
157:                 self.COTÉ*self.getligne(index)+self.COTÉ)]
158:             une_case.cousines = cousines_en_ligne
159:             cousines_en_colonne = [i for i in range(self.getcolonne(index),
160:                                                     self.NBR_CASES, self.COTÉ)]
161:             une_case.cousines.extend(cousines_en_colonne)
162:             cousines_en_bloc = list()
163:             if self.getbloc(index) < 3:
164:                 première_triplette = [i for i in range(
165:                     self.getbloc(index)*3, self.getbloc(index)*3+3)]
166:             elif self.getbloc(index) < 6:
167:                 première_triplette = [i for i in range(
168:                     18+self.getbloc(index)*3, 21+self.getbloc(index)*3)]
169:             else:
170:                 première_triplette = [i for i in range(
171:                     36+self.getbloc(index)*3, 39+self.getbloc(index)*3)]
172:             for i in range(3):
173:                 cousines_en_bloc.extend(list(map(lambda x: x+9*i,
174:                                                  première_triplette)))
175:             une_case.cousines.extend(cousines_en_bloc)
176:             liste_cases.append(une_case)
177:         self.contenu = liste_cases
178: 
179:     def __getitem__(self, index):
180:         """
181:         Permet d'obtenir le contenu d'une case de la grille avec:
182:         ma_grille[index] # où index est compris entre 0 et NBR_CASES-1.
183:         """
184:         return self.contenu[index]
185: 
186:     def __setitem__(self, index, symbole):
187:         """
188:         Permet d'écrire dans le contenu d'une case de la grille.
189: 
190:         exemple:
191:         -------
192:         ma_grille[0] = 5
193:         """
194:         if index < self.NBR_CASES:
195:             self.contenu[index].contenu = symbole
196:         else:
197:             raise IndexError()
198: 
199:     def __len__(self):
200:         return self.NBR_CASES
201: 
202:     def __repr__(self):
203:         """
204:         Affichage d'une grille.
205: 
206:         Lorsque l'on tape son nom dans l'interpréteur
207:         son `contenu` est affiché sous forme d'une grille 9 x 9.
208:         """
209:         affichage = ""
210:         index = 0
211:         for une_case in self.contenu:
212:             affichage += une_case.__repr__()  # ajout de l'affichage d'une case
213:             if index % 27 == 26 and index < 80:
214:                 affichage += "\n\n"  # à faire toutes les 3 lignes
215:             elif index % 9 == 8:
216:                 affichage += "\n"  # sinon à faire toutes les lignes
217:             elif index % 3 == 2:
218:                 affichage += "  "   # sinon à faire toutes les 3 colonnes
219:             elif index % 9 in [0, 1, 3, 4, 6, 7]:
220:                 affichage += " "
221: 
222:             index += 1
223:         return affichage
224: 
225:     def remplir_case_avec(self, index, valeur):
226:         """
227:         Rempli la case d'index compris entre 0 et 80 avec `valeur`.
228:         """
229:         if (self.écriture_autorisée(index)):
230:             self.__setitem__(index, valeur)
231:             self.réduire_prétendants
232:             self.réduire_sac
233: 
234:     def écriture_autorisée(self, index):
235:         return self.autorisation_colonne(index)\
236:             and self.autorisation_ligne(index)\
237:             and self.autorisation_carré(index)
238: 
239:     def autorisation_colonne(self, index):
240:         return True
241: 
242:     def autorisation_ligne(self, index):
243:         return True
244: 
245:     def autorisation_carré(self, index):
246:         return True
247: 
248:     def réduire_prétendants(self, index):
249:         pass
250: 
251:     def réduire_sac(self, index):
252:         pass
253: 
254:     def getcolonne(self, index):
255:         """
256:         Retourne le numéro de colonne de la case d'index compris entre 0 et 80.
257: 
258:         Les 9 colonnes sont numérotées de 0 à 8.
259:         """
260:         return index % 9
261: 
262:     def getligne(self, index):
263:         """
264:         Retourne le numéro de ligne de la case d'index compris entre 0 et 80.
265: 
266:         Les 9 lignes sont numérotées de 0 à 8.
267:         """
268:         return index//9
269: 
270:     def getbloc(self, index):
271:         """
272:         Retourne le numéro du bloc 3 x 3 auquel appartient la case d'index
273:         compris entre 0 et 80.
274: 
275:         Il y a 9 blocs 3 x 3 d'index compris entre 0 et 8.
276:         """
277:         return self.getcolonne(index)//3 + (self.getligne(index)//3)*3
278: 
279:     def marquer_cousines(self, index):
280:         """
281:         Montre les cases cousines de la case d'index donné.
282:         """
283:         for cousine in self.contenu[index].cousines:
284:             self.remplir_case_avec(cousine, "*")
