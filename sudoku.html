2 Sacs de chiffres restant à placer.

 1: class sac:
 2:     """
 3:     Classe représentant un sac. Un sac contient de 0 à 9 numéros identiques.
 4: 
 5:     Un sac contient à l'origine 9 numéros identiques.
 6:     À l'initialisation de la grille, un certain nombre de ces numéros est
 7:     prélevé pour les placer sur la grille.
 8:     Au cours du jeu, tant qu'il reste des numéros dans un sac, le joueur peut
 9:     en piocher pour les placer sur la grille.
10:     """
11:     def __init__(self, numero):
12:         """
13:         Constructeur de sac contenant 9 numéros identiques
14:         """
15:         self.id = numero
16:         self.contenu = 9
17: 
18:     def __repr__(self):
19:         """
20:         Représentation d'un sac lorsque l'on tape son nom dans l'interpréteur.
21: 
22:         Le contenu du sac est renvoyé.
23:         """
24:         if self.contenu == 0:
25:             return "sac vide"
26:         elif self.contenu == 1:
27:             return "contient 1 chiffre {}".format(self.id)
28:         else:
29:             return "contient {} chiffres {}".format(self.contenu, self.id)
30: 
31: 
32: def sacs_pleins():
33:     """
34:     Initialisation d'une liste contenant 9 sacs
35: 
36:     chaque sac contenant chacun 9 numéros identiques
37:     """
38:     liste_sacs = list()
39:     for i in range(1, 10):
40:         liste_sacs.append(sac(i))
41:     return liste_sacs
42: 
43: 
44: liste_sacs = sacs_pleins()

3 Cases

La grille comporte 81 cases. 

 1: class case:
 2:     """
 3:     Classe représentant une case.
 4: 
 5:     Chacune des 81 cases de la grille possède un identifiant `id` unique
 6:     allant de 0 à 80.
 7:     Une case non vide a un `contenu` qui est affiché quand on tape le nom
 8:     de la case dans l'interpréteur.
 9:     Une case vide à des prétendants (valeurs possibles de la case).
10:     """
11:     def __init__(self, numero):
12:         self.id = numero
13:         self.contenu = None
14:         self.pretendants = [1, 2, 3, 4, 5, 6, 7, 8, 9]
15: 
16:     def __repr__(self):
17:         """
18:         Représentation d'une case.
19: 
20:         Lorsque l'on tape son nom dans l'interpréteur
21:         son `contenu` est renvoyé.
22:         """
23:         if self.contenu is None:
24:             return "⛶"
25:         else:
26:             return "{}".format(self.contenu)
27: 
28: 
29: def cases_vides():
30:     """
31:     initialisation d'une liste contenant les 81 cases
32: 
33:     numérotées de 0 à 80
34:     """
35:     NBR_CASES = 81
36: 
37:     liste_cases = list()
38:     for i in range(NBR_CASES):
39:         liste_cases.append(case(i))
40:     return liste_cases
41: 
42: 
43: liste_cases = cases_vides()
